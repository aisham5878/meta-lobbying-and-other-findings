## 📋 About this Application

meta-lobbying-and-other-findings is a tool designed to help you explore detailed investigations into Meta Platforms’ influence operations. The findings are based entirely on publicly available records. These include government filings, lobbying disclosures, campaign finance data, corporate records, and archival sources.

This software presents the information clearly, allowing you to study the connections behind lobbying actions and efforts around regulations affecting social media and app stores.

You do not need any technical knowledge to use this application. It runs on Windows, organizes the data for easy navigation, and lets you follow the evidence step-by-step.

## 🔍 How to Use the Application

The interface breaks down the investigation into clear sections. Here’s how to explore the contents:

- **Lobbying Reports**: View summarized data and original lobbying filings.
- **Campaign Contributions**: See tables of financial support linked to key players.
- **Public Records**: Access scanned IRS filings, registrations, and archived web pages.
- **Timeline**: Browse events in chronological order to understand cause and effect.
- **Search Tool**: Enter keywords or names to find specific evidence quickly.

Click on any item to open detailed views with citations and source links. The app will help you trace information back to official documents.

## 🚩 About the Investigation

This project documents how Meta Platforms used various channels to influence laws regulating age verification. It shows how lobbying shifted the responsibility to Apple’s and Google’s app stores. Every detail comes from verifiable public documents, avoiding speculation.

- Sources include IRS 990 filings, Senate lobbying disclosures, state registrations, and more.
- Data is cross-checked with investigative journalism and public archives.

The application makes it easier to review these findings and understand the connections step-by-step.
